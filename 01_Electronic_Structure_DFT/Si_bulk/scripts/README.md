# Scripts

This directory contains minimal scripts used to post-process
VASP electronic structure calculations for bulk silicon.

## Contents

- `extract_econv.sh`  
  Extracts total energies from ENCUT convergence calculations.

- `extract_kconv.sh`  
  Extracts total energies from k-point convergence calculations.

- `plot_band.gp`  
  Gnuplot script for visualizing electronic band structures.

- `plot_dos.gp`  
  Gnuplot script for visualizing total density of states (DOS).

## Notes

The scripts are written as reusable templates for this project.
Large raw output files are not included in the repository.
